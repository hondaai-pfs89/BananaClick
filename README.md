# BananaClick
バナナクリック（α版）
バナナの仕分けがテーマ。バナナをクリックして得点を競うカジュアルゲームです

【プレイ方法】
画面に出てくるバナナをクリック。クリックしたバナナの種類で点数が異なります
黄色いバナナ：+1点
青いバナナ：-2点
金のバナナ：5点
制限時間30秒

【制作目的】
Unity/C#の基礎習得を目的として制作。
ランダム生成、クリック判定、スコア管理、シーン遷移などゲーム制作の基本機能を実装。
